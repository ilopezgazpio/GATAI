## Local Deployment of site (Debug)

- apt requirements

```
Ruby header files
Ruby-bundler
Jekyll
```

- bundle requirements
```
jekyll - gem install bundler jekyll
```

Finally, run

```
ruby -v
jekyll -v
bundle exec jekyll serve --port 4000
```
